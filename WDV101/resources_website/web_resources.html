Web Resources

  • W3Schools HTML
    • HTML Tutorial by W3Schools. W3Schools has easy-to-follow tutorials on HTML, CSS, and more. Includes a "Try it Yourself" editor tool that shows the code and results side-by-side.
  • learn-html.org
    • Learn HTML offers a free, interactive HTML tutorial aimed at teaching coding directly from the web browser. This focuses on the coding and does not include information on local installation and execution. This site is free and ad-funded.
  • Kinsta Blog on HTML
    • The Best Way to Learn HTML for Free (and Why You Should) is a entry blog by Salman Ravoof from June 7, 2022. It includes a brief definition of HTML, reasons to learn it, examples of jobs available to those with HTML knowledge, a short list of sites that offer free HTML tutorials, and more.
  • HTML.com
    • HTML.com offers a general overview of HTML with some background information as well. Includes a table of contents to quickly access the information you want, or scroll down to read everything in order.
  • freeCodeCamp.org
    • This site has a YouTube channel as well as a free interactive HTML course. The full "HTML for Beginners" video is 4 hours and available on YouTube.
  • Web Development Degrees and Careers
    • A brief overview of a guide to web development and careers within web development.
  • Web Development, AAS
    • Information on the Web Development AAS degree offerred at Des Moines Area Community College (DMACC). Click on the DMACC logo below to visit their website.
